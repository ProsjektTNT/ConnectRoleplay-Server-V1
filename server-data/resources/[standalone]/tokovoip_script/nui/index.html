טוקוויפ
טוקוויפ הוא פלאגין שמחליף את המערכת הרגילה של הדיבור, בשימוש ייחודי עם פלאגין לטימספיק.

הטוקוויפ מאפשר הרבה יותר אופציות ויכולות, הדברים שכעת הטוקוויפ מאפשר:
  • סאונד איכותי יותר
  • מערכת רדיו
  • מערכת שמיעה ייחודית לכל שחקן
  • שיחוות בפלאפון
  • רעשי רדיו
  • אפקטים לרדיו

טוקוויפ חייב להיות מותקן ו רץ כדי לשחק בשרת

אם אין באפשרותך להכנס לערוץ בשרת טימספיק שלנו בבקשה הצטרף לערוץ תמיכה בטימספיק, כדי שחבר תמיכה/צוות יוכל לסייע לך.

You can find more information and download links at: http://forums.rmog.us
